# 重复标签页清理器 Chrome 扩展

## 📁 文件说明

这个文件夹包含了完整的Chrome扩展程序，可以直接安装使用。

### 核心文件
- `manifest.json` - 扩展配置文件
- `background.js` - 后台服务脚本
- `popup.html` - 弹出窗口界面
- `popup.js` - 弹出窗口脚本

### 安装步骤

1. **下载整个 `chrome-duplicate-window-cleaner` 文件夹到你的电脑**

2. **打开Chrome浏览器，进入扩展管理页面**
   - 在地址栏输入：`chrome://extensions/`

3. **启用开发者模式**
   - 点击右上角的"开发者模式"开关

4. **加载扩展**
   - 点击"加载已解压的扩展程序"
   - 选择下载的 `chrome-duplicate-window-cleaner` 文件夹
   - 点击"选择文件夹"

5. **开始使用**
   - 扩展会出现在工具栏上
   - 点击扩展图标打开清理界面

## 🚀 功能特点

- ✅ 检测并关闭重复的标签页
- ✅ 支持清理当前窗口或所有窗口
- ✅ 跳过正在播放音频的标签页
- ✅ 详细的清理统计信息
- ✅ 现代化UI设计
- ✅ 简洁易用的界面

## 📋 使用说明

1. 点击扩展图标打开清理界面
2. 选择"清理当前窗口重复标签页"或"清理所有窗口重复标签页"
3. 查看清理结果和统计信息
